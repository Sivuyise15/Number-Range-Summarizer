# Number-Range-Summarizer

Hi there!

This a Number Range Summarizer basic project to produce a comma delimited list of numbers, grouping the numbers into a range when they are sequential.

**How to run this?**

1. CLONE the repo.
2. Navigate to **Number-Range-Summarizer** directory
3. With the command "**make test**" - you can run the unit testing, with the command "**make run**" - you can manually test the program

**What you need to successfully run and test this program?**

1. Atleast Java 8
2. A terminal that support the use of the "**make**" command - preferable, Ubuntu
3. Alternative to step 2, an IDE that support java code execution.
